# sing-box 使用教程：订阅链接导入、节点测速与系统代理设置

适用平台：Android

适用关键词：sing-box 教程、sing-box Android 配置、sing-box 订阅链接。

本教程用于帮助用户把服务商提供的订阅链接导入 sing-box，完成节点测速，并选择可用节点。请在当地法律法规和服务条款允许的范围内使用网络代理工具。

## 操作步骤

### 1. 打开 Profiles

进入 Profiles 配置页，点击右下角添加按钮。

<img src="../../assets/images/tutorial/29.png" alt="sing-box 打开 Profiles截图" width="360">

### 2. 选择手动创建

在导入方式中选择 Create Manually。

<img src="../../assets/images/tutorial/30.png" alt="sing-box 选择手动创建截图" width="360">

### 3. 进入编辑页

进入配置编辑页面后，准备填写远程订阅信息。

<img src="../../assets/images/tutorial/31.png" alt="sing-box 进入编辑页截图" width="360">

### 4. 切换 Remote

将 Type 改为 Remote，在 URL 粘贴订阅链接，Name 填写便于识别的名称。

<img src="../../assets/images/tutorial/32.png" alt="sing-box 切换 Remote截图" width="360">

### 5. 确认配置下载

保存后确认订阅配置已经出现在列表中。

<img src="../../assets/images/tutorial/33.png" alt="sing-box 确认配置下载截图" width="360">

### 6. 开启订阅

点击右下角按钮启用当前配置。

<img src="../../assets/images/tutorial/34.png" alt="sing-box 开启订阅截图" width="360">

### 7. 测试并选择节点

进入 Groups，点击闪电按钮测试延迟，选择有延迟的节点。

<img src="../../assets/images/tutorial/35.png" alt="sing-box 测试并选择节点截图" width="360">

## 使用建议

- sing-box 对订阅格式更敏感，遇到导入失败时优先确认服务商是否支持 sing-box 格式。
