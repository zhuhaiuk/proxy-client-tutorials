# Clash Verge 使用教程：订阅链接导入、节点测速与系统代理设置

适用平台：Windows

适用关键词：Clash Verge 教程、Clash Verge 订阅导入、Windows Clash Verge 配置。

本教程用于帮助用户把服务商提供的订阅链接导入 Clash Verge，完成节点测速，并选择可用节点。请在当地法律法规和服务条款允许的范围内使用网络代理工具。

## 操作步骤

### 1. 启用服务模式

在设置中开启服务模式，保证系统代理能正常接管流量。

![Clash Verge 启用服务模式截图](../../assets/images/tutorial/75.png)

### 2. 导入订阅

在配置页的订阅文件链接处粘贴订阅链接，点击导入，并将新配置设为活动文件。

![Clash Verge 导入订阅截图](../../assets/images/tutorial/76.png)

### 3. 确认导入成功

看到配置文件出现在列表中后，说明订阅已经保存。

![Clash Verge 确认导入成功截图](../../assets/images/tutorial/77.png)

### 4. 开启系统代理

回到设置页面开启系统代理。

![Clash Verge 开启系统代理截图](../../assets/images/tutorial/78.png)

### 5. 选择节点

进入代理页面，选择一个有延迟的节点使用。

![Clash Verge 选择节点截图](../../assets/images/tutorial/79.png)

## 使用建议

- 旧版 Clash Verge 已归档，优先使用仍在维护的 Clash Verge Rev。
