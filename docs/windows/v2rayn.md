# V2rayN 使用教程：订阅链接导入、节点测速与系统代理设置

适用平台：Windows

适用关键词：V2rayN 教程、V2rayN 订阅分组、Windows V2rayN 配置。

本教程用于帮助用户把服务商提供的订阅链接导入 V2rayN，完成节点测速，并选择可用节点。请在当地法律法规和服务条款允许的范围内使用网络代理工具。

## 操作步骤

### 1. 打开订阅分组设置

点击订阅分组，选择订阅分组设置。

![V2rayN 打开订阅分组设置截图](../../assets/images/tutorial/97.png)

### 2. 添加分组

在订阅分组设置中点击添加。

![V2rayN 添加分组截图](../../assets/images/tutorial/98.png)

### 3. 填写订阅地址

别名填写备注，可选地址/url 粘贴订阅链接，点击确定保存。

![V2rayN 填写订阅地址截图](../../assets/images/tutorial/99.png)

### 4. 更新订阅

选择更新当前订阅；如果订阅地址被网络阻断，再尝试通过代理更新。

![V2rayN 更新订阅截图](../../assets/images/tutorial/100.png)

### 5. 确认更新结果

节点列表出现后说明订阅已下载。

![V2rayN 确认更新结果截图](../../assets/images/tutorial/101.png)

### 6. 测试真连接

Ctrl+A 全选节点，右键选择测试服务器真连接延迟，使用有延迟的节点。

![V2rayN 测试真连接截图](../../assets/images/tutorial/102.png)

## 使用建议

- V2rayN 的“真连接延迟”比普通 ping 更接近实际可用性。
