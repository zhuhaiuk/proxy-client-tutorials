# Hiddify for Windows 使用教程：订阅链接导入、节点测速与系统代理设置

适用平台：Windows

适用关键词：Hiddify Windows 教程、Hiddify 订阅设置、Windows Hiddify 配置。

本教程用于帮助用户把服务商提供的订阅链接导入 Hiddify for Windows，完成节点测速，并选择可用节点。请在当地法律法规和服务条款允许的范围内使用网络代理工具。

## 操作步骤

### 1. 新建配置

在主界面点击添加新的配置文件。

![Hiddify for Windows 新建配置截图](../../assets/images/tutorial/104.png)

### 2. 选择手动输入

在添加方式中选择手动输入。

![Hiddify for Windows 选择手动输入截图](../../assets/images/tutorial/105.png)

### 3. 填写订阅链接

在网址处粘贴订阅链接，名称填写备注，然后点击保存。

![Hiddify for Windows 填写订阅链接截图](../../assets/images/tutorial/106.png)

### 4. 确认保存

看到配置保存成功后返回主界面。

![Hiddify for Windows 确认保存截图](../../assets/images/tutorial/107.png)

### 5. 选择可用节点

进入代理页，等待 Hiddify 自动测速，选择有延迟的节点。

![Hiddify for Windows 选择可用节点截图](../../assets/images/tutorial/108.png)

## 使用建议

- Hiddify 自动化程度较高，适合不想手动维护太多规则的用户。
