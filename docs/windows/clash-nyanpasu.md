# Clash Nyanpasu 使用教程：订阅链接导入、节点测速与系统代理设置

适用平台：Windows

适用关键词：Clash Nyanpasu 教程、Nyanpasu 订阅导入、Windows Mihomo 客户端。

本教程用于帮助用户把服务商提供的订阅链接导入 Clash Nyanpasu，完成节点测速，并选择可用节点。请在当地法律法规和服务条款允许的范围内使用网络代理工具。

## 操作步骤

### 1. 打开服务模式

进入设置，开启服务模式。

![Clash Nyanpasu 打开服务模式截图](../../assets/images/tutorial/82.png)

### 2. 导入订阅

打开配置页面，在配置文件链接处粘贴订阅链接并点击导入。

![Clash Nyanpasu 导入订阅截图](../../assets/images/tutorial/83.png)

### 3. 确认导入

看到导入成功提示或新配置文件后继续。

![Clash Nyanpasu 确认导入截图](../../assets/images/tutorial/84.png)

### 4. 设为活动订阅

点击刚导入的订阅，将其设置为活动配置。

![Clash Nyanpasu 设为活动订阅截图](../../assets/images/tutorial/85.png)

### 5. 选择节点

返回代理页面，选择有延迟的节点使用。

![Clash Nyanpasu 选择节点截图](../../assets/images/tutorial/86.png)

## 使用建议

- 如果代理页没有延迟结果，先手动触发一次节点测速。
