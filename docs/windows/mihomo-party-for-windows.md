# Mihomo Party for Windows 使用教程：订阅链接导入、节点测速与系统代理设置

适用平台：Windows

适用关键词：Mihomo Party 教程、mihomo-party 订阅导入、Windows Mihomo 客户端。

本教程用于帮助用户把服务商提供的订阅链接导入 Mihomo Party for Windows，完成节点测速，并选择可用节点。请在当地法律法规和服务条款允许的范围内使用网络代理工具。

## 操作步骤

### 1. 导入订阅

点击空白订阅，在订阅地址输入栏粘贴订阅链接，点击导入。

![Mihomo Party for Windows 导入订阅截图](../../assets/images/tutorial/119.png)

### 2. 确认导入

看到订阅出现在列表中，说明配置已经添加。

![Mihomo Party for Windows 确认导入截图](../../assets/images/tutorial/120.png)

### 3. 进入代理组

点击代理组位置，查看节点和策略组。

![Mihomo Party for Windows 进入代理组截图](../../assets/images/tutorial/121.png)

### 4. 测速选节点

展开节点信息，等待自动测速；如没有延迟，点击测速图标手动测试。

![Mihomo Party for Windows 测速选节点截图](../../assets/images/tutorial/122.png)

### 5. 开启系统代理

点击开启系统代理，即可开始使用。

![Mihomo Party for Windows 开启系统代理截图](../../assets/images/tutorial/123.png)

## 使用建议

- Mihomo Party 适合偏新内核的 Windows 代理客户端使用场景。
