# NekoBox for Windows 使用教程：订阅链接导入、节点测速与系统代理设置

适用平台：Windows

适用关键词：NekoBox Windows 教程、NekoRay 订阅设置、Windows NekoBox 配置。

本教程用于帮助用户把服务商提供的订阅链接导入 NekoBox for Windows，完成节点测速，并选择可用节点。请在当地法律法规和服务条款允许的范围内使用网络代理工具。

## 操作步骤

### 1. 打开分组

点击首选项，选择分组。

![NekoBox for Windows 打开分组截图](../../assets/images/tutorial/89.png)

### 2. 新建分组

在分组窗口点击新建分组。

![NekoBox for Windows 新建分组截图](../../assets/images/tutorial/90.png)

### 3. 填写订阅

类型选择订阅，名称填写备注，勾选手动调节列宽，在 URL 粘贴订阅链接。

![NekoBox for Windows 填写订阅截图](../../assets/images/tutorial/91.png)

### 4. 更新订阅

保存后点击更新订阅。

![NekoBox for Windows 更新订阅截图](../../assets/images/tutorial/92.png)

### 5. 确认节点

看到节点出现在列表中，说明订阅更新成功。

![NekoBox for Windows 确认节点截图](../../assets/images/tutorial/93.png)

### 6. 测速并开启代理

选择订阅分组，右键测试真连接延迟，选择可用节点并开启系统代理。

![NekoBox for Windows 测速并开启代理截图](../../assets/images/tutorial/94.png)

## 使用建议

- NekoBox/NekoRay 的菜单较多，订阅链接一定要填在分组 URL 中。
